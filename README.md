# Projekt_PA_UAR

## Opening the project:
- Install needed libraries:
 - `flask`  
- When every python library is installed, all you need to do is to type in the repository location:
```bash
  $env:FLASK_APP= "main" #in powershell
  export FLASK_APP=main.py #in bash
  #and
  flask run
```
